# Screenshot Studio

A modern, browser-based screenshot editor for creating beautiful screenshots, social media assets, tutorials, presentations, and product demos.

Live : https://www.screenshot-studio.com/

## ✨ Features

### 🎨 Powerful Screenshot Editing

* Draw with arrows, shapes, text, and blur tools
* Crop, resize, rotate, and transform screenshots
* Fine-grained styling controls for professional results

### 🖼️ Backgrounds

* **100+ built-in backgrounds**
* Beautiful gradients
* Solid colors
* Image backgrounds
* Blur effects
* Noise textures

### 🌐 Browser Mockups

* Safari (Light & Dark)
* Chrome (Light & Dark)
* Adjustable browser header
* Realistic browser toolbars
* Custom URL support

### 💻 Device & Frame Styles

* Arc Browser
* Polaroid
* Glass
* Outline
* Border
* Multiple presentation styles

### 🧊 3D Perspective

* **30+ perspective presets**
* Realistic depth effects
* Interactive transforms
* Modern presentation styles

### ✏️ Draw & Markup

* Arrows
* Shapes
* Blur regions
* Text overlays
* Highlight important areas

### 🐦 Content Generation

* Import and style Tweets
* Generate beautiful code snippets
* Developer-friendly sharing tools

### 🎬 Animations

* **20+ animation presets**
* Timeline editor
* Keyframe controls
* Smooth transitions

### 🎥 Export Options

* PNG
* JPG
* MP4
* WebM
* GIF

### ⚡ High Performance

* High-resolution exports up to **5× scale**
* Hardware-accelerated video encoding
* Fully browser-based
* No desktop application required

# 🚀 Getting Started

## Prerequisites

* Node.js 20+
* npm, pnpm, or Bun

## Installation

```bash
git clone git@github.com:Nova-cmd-oss/screenshot-studio-.git
cd screenshot-studio-
```

Install dependencies:

```bash
npm install
```

or

```bash
bun install
```

Start the development server:

```bash
npm run dev
```

or

```bash
bun run dev
```

Open **http://localhost:3000** in your browser.

# 🌟 Why Screenshot Studio?

Screenshot Studio is designed as an all-in-one editor for creating production-quality screenshots without relying on multiple tools. Whether you're preparing marketing assets, documentation, tutorials, social media posts, or product announcements, it combines editing, mockups, animations, and exporting into a single streamlined workflow.
