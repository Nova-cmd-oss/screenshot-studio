# Screenshot Studio - Free Online Screenshot Editor

## About
Screenshot Studio is a free, open-source, browser-based screenshot editor that transforms plain screenshots into professional-quality graphics. Your images never leave your browser. No signup, no watermarks, no downloads required.

## Core Features
- **100+ Backgrounds** — Gradients, mesh, radiant, paper textures, patterns, solid colors
- **Browser Mockups** — Safari and Chrome frames (light and dark mode), Arc browser
- **Device Frames** — macOS, Polaroid, glass, outline, border
- **3D Effects** — Perspective transforms, rotation, tilt, depth-of-field
- **Animation Timeline** — 20+ animation presets (zoom, pan, Ken Burns, tilt, fade, bounce, slide)
- **Video Export** — MP4, WebM, GIF via FFmpeg WASM (runs in browser)
- **Text Overlays** — 27+ Google Fonts, custom styling, shadows, positioning
- **Tweet Import** — Convert tweets to images by URL
- **Code Snippets** — Syntax-highlighted code to image
- **High-Res Export** — PNG, JPEG, WebP up to 5x resolution
- **Templates & Presets** — One-click professional styling

## Languages
Available in 8 languages: English, Spanish (/es), French (/fr), German (/de), Japanese (/ja), Portuguese (/pt), Korean (/ko), Chinese (/zh).

## Technology Stack
- Next.js 16 with App Router
- React 19
- TypeScript
- HTML Canvas with modern-screenshot
- Cloudflare R2 for asset storage
- Tailwind CSS, Radix UI
- Zustand for state management
- FFmpeg WASM for video encoding
- next-intl for internationalization

## Use Cases
- Social media graphics (Twitter/X, Instagram, LinkedIn, Product Hunt)
- SaaS product screenshots for landing pages and documentation
- Developer portfolio and project showcase images
- Marketing presentations and pitch decks
- Animated slideshows and video content

## Target Audience
Developers, designers, marketers, indie hackers, content creators, and anyone who needs professional visuals without design skills.

## Pricing
100% free. No signup, no watermarks, no hidden costs, no premium tier. Unlimited exports with full feature access.
